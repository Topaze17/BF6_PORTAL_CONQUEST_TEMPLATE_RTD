### BF6 Conquest Template RTD
This GitHub project aims to provide a high-fidelity Conquest Template for the Battlefield Portal environment. It is designed for creators who want to bridge the gap between custom experiences and the standard BF6 game mode.
the principal aim of this template is :
 - Maximum Fidelity: To replicate the "Normal Conquest" experience as closely as possible within the constraints of the Portal Logic Editor.
 - Performance Optimization: Built with a "Fidelity First" mindset, optimized to support at least 32 players + AI, with a target goal of maintaining stability for 64-player matches.
 - User-Friendly Implementation: Designed to be "Plug-and-Play." Map creators should be able to integrate this logic into their custom maps with minimal configuration.

**Why would I do a BF6 Conquest template as one already exists?**

I want to create this template because I know some people do not take the step to try out Portal maps due to the lack of visual fidelity compared to the normal Conquest mode. As such, I really want to create something as faithful as possible to the normal game mode and let this template be used by map creators. My goal is to allow everyone to enjoy those maps—even my more reticent friends!

**Everyone** is welcome to help! Please don't be shy—feel free to submit pull requests. I might not review them instantly, but your input will be invaluable!
